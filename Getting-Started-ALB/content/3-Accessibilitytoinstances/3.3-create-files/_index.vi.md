1. Chạy lệnh sau để tạo một test file:

```
touch test.txt
```

2. Chạy lệnh sau để xác minh file đã được tạo:

```
ls
```

Tiếp theo, chúng ta sẽ tiến hành **khôi phục EC2 từ AMI** để kiểm tra tính bền vững của dữ liệu.
